# Branding Assets

This folder is intentionally split into two layers:

- `current/` contains the selected live assets that the site, plugin listing, and public docs should reference.
- `archive/` contains historical logo exploration, comparison boards, and unused directions kept only for context.

Working rule:

- new public-facing references should point only at `current/`
- exploration should be preserved in `archive/`
- do not move an exploratory asset back into live use without updating the owning docs and surface references
